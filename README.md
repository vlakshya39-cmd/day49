# Personal AI Playbook

A browser-based Personal AI Playbook built during **Day 49** of the **#60DayClaudeChallenge**.

This application helps users organize reusable AI workflows, build prompts using modular blocks, convert prompts into autonomous loops, and export workflow libraries for backup and reuse.

---

## Features

- 📚 Workflow Library
- 📝 Prompt Builder
- 🔄 Loop Builder
- ⭐ Favorite Workflows
- 🔍 Workflow Search
- 💾 Save Custom Workflows
- 📤 Export Workflow Library (JSON)
- 📥 Import Workflow Library
- 🌙 Dark/Light Theme
- 📱 Responsive Design

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage

## How to Run

1. Download or clone the repository.
2. Open `personal-ai-playbook.html` in any modern web browser.
3. Create workflows.
4. Build prompts using Prompt Builder.
5. Convert prompts into autonomous loops.
6. Export your workflow library as JSON.
